  <ul>
  <li>Use Express JS to build the API.</li>
   <li>Use any port you want for the API.</li>

</ul>



```javascript
{
  "message": "pong"
}
```

  <li>Use Postman to test your API.</li>

  <ul>

  </ul>
</ul>
